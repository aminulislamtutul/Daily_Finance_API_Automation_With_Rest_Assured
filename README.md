# **Daily Finance API Automation With Rest Assured**
This project is focused on testing and automating the RESTful API endpoints of the [**Daily Finance**](https://dailyfinance.roadtocareer.net/) web application.
## Features Covered in Postman Collection
Using browser Developer Tools → Network tab, the following API interactions were captured and added to the Postman collection: [**Link Here**](https://gold-robot-3070368.postman.co/workspace/Md.-Aminul-Islam's-Workspace~85fe5b2b-35f5-4c6e-8ec4-625f56db4d81/collection/46763324-26a1eb2a-3bcf-49a6-b44a-da4723711efb?action=share&source=copy-link&creator=46763324) 
- User Registration
- Admin Login
- Fetch User List
- Search User by ID
- Edit User Info (e.g., first name, phone number)
- User Login
- Fetch Item List
- Add Item
- Edit Item Name
- Delete Item
